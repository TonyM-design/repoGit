REGLE DU JEU


Bienvenue dans ce jeu de plateau au tour par tour, un seul but : vaincre l'adversaire !
Dans ce jeu de duel les règles sont simples :
  • pouvez vous déplacer jusqu'à trois case par tour
  • à la fin de chaque deplacement vous pouvez vous battre contre un adversaire présent sur une case adjacentes
  • il est possible de se défendre pour reduire les dégat subit, cette action met fin à vos déplacement et vous empèche d'attaquer tant que la posture défensive est active
  • des armes sont présentes n'hésitez pas à vous en servir pour terraser votre adversaire
pour vous déplacer utilisez les flèches directionnelles de votre clavier, vous pouvez mettre fin à vos deplacement à tout moment en appuyant sur la touche espace et ainsi déclencher une phase de combat, enfin pour activer/desactiver la posture défensive, appuyez sur la touche 0 de votre pav. numerique
